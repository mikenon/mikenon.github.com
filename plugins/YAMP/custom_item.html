YAMP is a plugin for Spout that allows you to create custom blocks and items without the knowledge of java.

This page enables you to create a yml file that YAMP uses to create new blocks.

New: If your server uses JSONAPI you can now upload custom blocks directly to the YAMP directory

New: The Clear button at the bottom of the form lets you create multiple blocks without loading the page, maintaining the JSONAPI connection if present
